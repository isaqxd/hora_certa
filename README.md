# 🌐 Hora Certa 
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)[![All Contributors](https://img.shields.io/badge/all_contributors-7-orange.svg?style=flat-square)](#contributors-)

Este repositório faz parte da atividade de Teste de Software, contendo a documentação do sistema **Hora Certa** - uma solução desenvolvida para auxiliar idosos no gerenciamento autônomo de sua rotina de saúde.

## Documentação Completa
Toda a documentação do projeto, incluindo objetivos, regras de negócio, histórias de usuário e requisitos funcionais com critérios de aceitação em Gherkin, está disponível na **wiki do projeto**.
**[Acessar a Wiki](https://github.com/isaqxd/hora_certa/wiki)**

## Como navegar na Wiki
A wiki está organizada nas seguintes seções:
- [**Home**](https://github.com/isaqxd/hora_certa/wiki#objetivo-geral) - Objetivo Geral e Descrição do projeto.
- [**Glossário**](https://github.com/isaqxd/hora_certa/wiki/1.Gloss%C3%A1rio#gloss%C3%A1rio) - Visão geral do sistema Hora Certa
- [**Objetivos Específicos**](https://github.com/isaqxd/hora_certa/wiki/2.Objetivos-Espec%C3%ADficos) - Metas detalhadas do projeto
- [**Regras de Negocio**](https://github.com/isaqxd/hora_certa/wiki/3.Regras-de-Negocio) - Premissas e restrições do sistema
- [**Historias de Usuario**](https://github.com/isaqxd/hora_certa/wiki/4.Historias-de-Usuario
) - Necessidades dos usuários
- [**Requisitos Funcionais**](https://github.com/isaqxd/hora_certa/wiki/5.Requisitos-Funcionais) - Requisitos com critérios de aceitação em Gherkin
